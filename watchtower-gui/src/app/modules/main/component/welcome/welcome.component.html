Get started
Welcome in Nextflow Tower

Nextflow Tower allows you to monitor the execution of any of your Nextflow data analysis pipelines.

1. Download latest version of Nextflow

To use Nextflow Tower download the latest Nextflow snapshot.

2. Set the Tower access token in your environment
export TOWER_ACCESS_TOKEN={{user.data.userName}}:{{user.data.nfAccessToken}}
3. Run Nextflow with Tower option

Launch nextflow as usual adding the -with-tower command line option 

nextflow run hello -with-tower
Enjoy it!
