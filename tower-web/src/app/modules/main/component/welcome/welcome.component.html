Get started
Welcome in Nextflow Tower

Nextflow Tower allows you to monitor the execution of any of your Nextflow data analysis pipelines.

1. Download latest version of Nextflow

Download latest Nextflow from this link. Rename it as `nextflow` and give it execute permission as shown below: 

chmod +x nextflow
2. Set the access token

Define the following environment variable in your environment: 

export TOWER_ACCESS_TOKEN={{user.data.nfAccessToken}}
3. Run Nextflow

Launch nextflow as usual adding the -with-tower command line option: 

./nextflow run hello -with-tower
Enjoy it

The pipeline execution will popup here!
